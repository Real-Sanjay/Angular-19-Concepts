What Are Custom Pipes in Angular?

How to Create a Custom Pipe

Step 2: Define the Pipe

import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'appendText'
})
export class AppendTextPipe implements PipeTransform {
  transform(value: string, suffixVal: string): string {
    return value + suffixVal;
  }
}

MCQ  Questions
Q1: What is the purpose of the @Pipe decorator?
A: It provides metadata to Angular, such as the name of the pipe, and allows the pipe to be registered and used in templates.

Q2: What is the role of the PipeTransform interface?
A: It enforces the implementation of the transform() method, which contains the transformation logic for a custom pipe.

Q3: Can a pipe take multiple arguments?
A: Yes. The transform() method can accept multiple parameters beyond the initial value.

Q4: Can you chain custom pipes like built-in pipes?
A: Yes, just like built-in pipes, custom pipes can be chained.
